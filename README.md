# Каталог Музея

## Описание предметной области

Информационная система «Каталог Музея» предназначена для учёта, хранения и управления данными о музейных экспонатах, а также для обеспечения работы сотрудников музея и предоставления информации посетителям.

### Ключевые сущности системы:

| Сущность | Описание | Количество записей |
|----------|----------|-------------------|
| **Экспонаты (Exhibitss)** | Предметы искусства и истории с информацией о названии, инвентарном номере, датировке, материале изготовления, авторах, текущем местоположении и оценочной стоимости | **120** |
| **Экспозиции (Expositions)** | Тематические выставки с описанием, датами проведения и кураторами | **25** |
| **Залы и хранилища (HallsStorages)** | Помещения для размещения экспонатов с указанием условий хранения (температура, влажность, площадь) | **25** |
| **Авторы и создатели (AuthorsCreators)** | Художники, скульпторы, мастера с биографическими данными (страна, годы жизни, биография) | **25** |
| **История поступления (AdmissionHistory)** | Информация о способах приобретения экспонатов: покупка, дар, археологическая находка, передача, конфискация | **25** |
| **Пользователи (Users)** | Учётные записи сотрудников системы с разграничением прав доступа | **6** |
| **Роли (Roles)** | Роли пользователей: Администратор, Менеджер, Пользователь | **3** |
| **Журнал реставраций (RestorationLogs)** | История реставрационных работ по экспонатам | **30** |

### Бизнес-процессы:

- ✅ Учёт состояния и реставрации экспонатов
- ✅ Планирование перемещений между выставками и залами
- ✅ Формирование цифровых коллекций для сайта музея
- ✅ Управление экскурсионными программами
- ✅ Отчётность перед министерством культуры
- ✅ Разграничение прав доступа по ролям пользователей (Администратор, Менеджер, Пользователь)
- ✅ Авторизация и регистрация пользователей системы
